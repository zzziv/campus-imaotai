## 项目介绍

i茅台app，每日自动预约茅台

- [x] 平台注册账号(可用i茅台)
- [x] 添加多个用户
- [x] 自动预约
- [x] 类型选择（本市出货量最大的门店，或位置附近门店）
- [x] 自动旅行
- [x] 首次旅行分享
- [x] 获取申购耐力值
- [x] 自定义时间/随机时间预约或旅行
- [x] 结果消息推送

此项目使用 **Campus** 进行编写：<https://github.com/oddfar/campus>
